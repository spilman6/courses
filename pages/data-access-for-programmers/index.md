# Data Access for Programmers

> Learn relational data modeling, SQL querying, and API-backed data workflows.

## Weekly Roadmap

| Week | Topic | Lesson |
|---|---|---|
| **Week 1** | **Course setup and workflow**<br>You’ll learn about basic data types. | [Getting Started](/spilman/data-access-for-programmers/getting-started) |
| **Week 2** | **Data structures and storage formats**<br>You’ll learn how programs represent and store different kinds of data. | [Objects and File Types](/spilman/data-access-for-programmers/objects-and-file-types) |
| **Week 3** | **Intro to relational databases**<br>You’ll learn how single-table databases are structured and used. | [Single Table Databases](/spilman/data-access-for-programmers/single-table-databases) |
| **Week 4** | **Core SQL query patterns**<br>You’ll learn to retrieve and filter records with foundational SQL queries. | [Single Table Queries](/spilman/data-access-for-programmers/single-table-queries) |
| **Week 5** | **Aggregation and grouped analysis**<br>You’ll learn to summarize data using GROUP BY and HAVING. | [Group by and Having Statements](/spilman/data-access-for-programmers/group-by-and-having-statements) |
| **Week 6** | **Parent-child data modeling**<br>You’ll learn how one-to-many relationships connect related data. | [One-to-Many Relationships](/spilman/data-access-for-programmers/one-to-many-relationships) |
| **Week 7** | **Querying across related tables**<br>You’ll learn to combine related data with multi-table query techniques. | [Multi-Table Queries](/spilman/data-access-for-programmers/multi-table-queries) |
| **Week 9** | **Advanced SQL techniques**<br>You’ll learn advanced joins, views, and stored procedures. | [Advanced Joins, Views, and Procedures](/spilman/data-access-for-programmers/advanced-joins-views-and-procedures) |
| **Week 10** | **Many-to-many design patterns**<br>You’ll learn how to model many-to-many relationships correctly. | [Many-to-Many Relationships](/spilman/data-access-for-programmers/many-to-many-relationships) |
| **Week 11** | **Schema quality and optimization**<br>You’ll learn normalization techniques to improve database design. | [Database Normalization](/spilman/data-access-for-programmers/database-normalization) |
| **Week 12** | **API fundamentals for data apps**<br>You’ll learn how RESTful APIs connect applications to data. | [RESTful APIs](/spilman/data-access-for-programmers/restful-apis) |

## Course Notes

- Complete each week in order unless your instructor says otherwise.
- Bring questions to class with the week number and topic for faster support.
